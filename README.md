# The Genesys Blockchain

A Blockchain based application written in Python to run it's own crypto currency and exchange.

## Requirements

- Python3

## Create a Virtual Environment

```sh
python3 -m venv venv
```

## Acitivate venv

_Linux_

```sh
source venv/bin/activate
```

_Windows_

```sh
venv\Scripts\activate.bat
```
